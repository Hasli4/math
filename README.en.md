# Math Trainer

[Русская версия](README.md)

An interactive website for math practice and clear step-by-step explanations. The project is designed as a practical tool for teachers: you can quickly build an example, show the solution step by step, and use the page during lessons or homework.

## What the site can do

- Fractions:
  addition, subtraction, multiplication, and division with step-by-step explanations.
- Percentages:
  finding a percentage of a number and finding a number from a known percentage.
- Column arithmetic:
  addition, subtraction, multiplication, and division in the school-style written format.
- Word problems:
  image upload, drawing on top of the image, text labels, shapes, and staged solution notes.

## What it is for

The site is meant not only to give an answer, but to show the full reasoning:

- what the original example was;
- how the example changes at each step;
- where intermediate numbers come from;
- how to explain the solution clearly to a student.

This is especially useful in lessons where the explanation matters as much as the final answer.

## Who this project is for

This project is useful for:

- math and computer science teachers;
- one-on-one lessons;
- online classes;
- independent student practice.

## Project idea

The main goal is to create a clear teaching tool where the solution is revealed step by step instead of showing only the final answer.
