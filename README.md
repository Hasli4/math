# Математический тренажер

[English version](README.en.md)

Интерактивный сайт для тренировки математики и наглядного разбора решений. Проект задуман как удобный инструмент для преподавателя: можно быстро собрать пример, показать решение по шагам и использовать страницу на уроке или в домашней работе.

## Что умеет сайт

- Дроби:
  сложение, вычитание, умножение и деление дробей с пошаговым объяснением.
- Проценты:
  поиск процента от числа и поиск числа по проценту.
- Столбик:
  сложение, вычитание, умножение и деление в школьной записи.
- Задачи:
  загрузка изображения, рисование поверх него, подписи, фигуры и поэтапный разбор решения.

## Для чего это сделано

Сайт помогает не просто получить ответ, а показать ход мысли:

- что было в исходном примере;
- как пример меняется на каждом шаге;
- откуда берутся промежуточные числа;
- как красиво объяснить решение ученику.

Это особенно полезно на занятиях, где важно разбирать тему спокойно и наглядно.

## Как запустить

Это статический проеект, опубликован на GitHub Pages, можно открыть по ссылке - [https://hasli4.github.io/math/](https://hasli4.github.io/math/)

## Основные файлы

- `index.html` — структура страницы и вкладок;
- `styles.css` — оформление;
- `app.js` — логика тренажеров;
- `logo/` — favicon и иконки сайта.

## Для кого проект

Проект подходит для:

- преподавателей математики и информатики;
- индивидуальных занятий;
- онлайн-уроков;
- самостоятельной тренировки учеников.

## Идея проекта

Главная цель — сделать понятный учебный инструмент, где решение можно показывать по шагам, а не просто выдавать готовый ответ.
